### 2.3 Layout Foundation

**Body:**
```css
html {
    color-scheme: light only;
}

body {
    font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif;
    background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
    color: #333;
    padding: 20px;
    min-height: 100vh;
    line-height: 1.6;
    margin: 0;
}
```

**Browser dark mode opt-out:** The `html { color-scheme: light only; }` rule (paired with `<meta name="color-scheme" content="only light">` in `<head>`) prevents browsers from applying automatic dark mode transformations to the quote HTML. See Section 5.10 in [AGENTS.QUOTING.md](../AGENTS.QUOTING.md) for the full rule.

**Container:**
```css
.container {
    max-width: 1400px;
    margin: 0 auto;
    background: white;
    color: #333;
    border-radius: 15px;
    box-shadow: 0 20px 60px rgba(0,0,0,0.3);
    overflow: hidden;
}
```

**Section spacing:**
- Sections: `margin: 20px 30px`
- First section: `margin-top: 30px`
- Last section: `margin-bottom: 30px`
- Section content padding: `25px 30px`
- Lists in section content: `margin: 8px 0 16px 25px`
- List items: `margin-bottom: 6px`

**Responsive breakpoint** at `768px`:
```css
@media (max-width: 768px) {
    header h1 { font-size: 2em; }
    .info-grid { grid-template-columns: 1fr; }
    .resp-grid { grid-template-columns: 1fr; }
    .signature-grid { grid-template-columns: 1fr; }
    .section { margin: 12px 15px; }
    .section-content { padding: 18px; }
}
```

**Print styles:**
```css
@media print {
    body { background: white; padding: 0; }
    .container { box-shadow: none; border-radius: 0; }
    .section { margin: 10px 20px; break-inside: avoid; }
    .section-header::after { display: none; }
    .section.collapsed .section-content { display: block !important; }
    .phase-card:hover { transform: none; box-shadow: none; }
}
```
